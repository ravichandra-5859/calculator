# 🧮 Calculator App

A modern and responsive Calculator App built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean UI, colorful buttons, hover effects, and mobile-friendly design. This project improves DOM manipulation, event handling, and frontend development skills.

---

## 🚀 Features

- Addition, Subtraction, Multiplication, Division
- AC (Clear All) button
- Delete Last Character
- Responsive Design
- Smooth Hover Effects
- Modern UI Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
